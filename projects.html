Individual Tax Return Filling

Fundamental Taxation

This is my first individual accounting project. I was given a fictitious case and asked to file a tax return package for client. The project contains two parts: the tax returns and a written document explaining all decisions I made. This project gave me a chance to practice all the taxation regulations and filling techniques that I learned during the course.

To be successful, I need to analyze client’s source of income and investments in order to declare correct taxable portions; yet, still able to help client saves on possible deductible. I also learned how to use tax filling software while working on the project. This is a good way to connect my study with real world scenarios.

Resource: Internal Revenue Service's Official Website.

Xerox Project

Analysis and Design of Accounting Systems

This project required some professional judgements and technical skills as an accountant. I was given a year – end financial statements with hidden errors. My job was to identify the errors and make corrections in the final financial report. This is a lengthy and complex project, which require details analysis. I had to use excel and accounting software Xerox in order to fulfill all the requirements.

Despite its difficult level, this project is my favorite because it represents the work and challenge that accountants have to deal with. The project gave me a chance to be familiar with actual accounting tasks and helped to prepare myself for future career.

Business Give Back

Business Communication

This is a group project for business communication class called “Business Give Back Competition”. For this project, each team had to represent a company and develop a business plan to address a social issue in local area. My team chose to raise awareness about excessive hydraulic fracturing in Dallas area and its effect on the environment. The project consists of two portions: a detail business plan and a presentation.

This project is helpful for every business student because it helps to improve many essential skills including working in team, professional writing, critical thinking, and public speaking. At the end, my group placed first in class thanks to amazing contributions from every member.
